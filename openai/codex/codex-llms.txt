# Codex

> The purpose of the Codex CLI is to provide a locally installed, command-line interface for interacting with OpenAI’s Codex AI agent. It allows users to execute code, generate code, and interact with the agent in a terminal environment. The CLI supports features like sandboxing, execpolicy for command control, authentication via ChatGPT, and integration with IDEs. It also offers advanced capabilities such as model context protocol (MCP) for server-side execution, zero data retention (ZDR), and extensibility via SDKs and GitHub Actions. The goal is to empower developers to use powerful AI coding assistance locally, with full control over execution and privacy.

**Remember:**
- Codex CLI: Command-line interface for interacting with OpenAI’s local AI coding agent
- Model Context Protocol (MCP): Protocol for executing commands via remote servers
- Execpolicy: Rules governing which commands Codex can execute
- Sandboxing: Isolation of execution to prevent system-wide risks
- Zero Data Retention (ZDR): Ensures no user data is stored on servers
- Authentication via ChatGPT: Integration with OpenAI’s ChatGPT account for access
