# Langgraphjs

> LangGraph.js empowers developers to build robust, controllable AI agents capable of handling complex, multi-step tasks with long-term memory and human-in-the-loop interactions. It provides low-level primitives for custom agent design, supports streaming for real-time reasoning, and integrates with LangChainâ€™s ecosystem for seamless LLM application development. The project also offers enterprise-grade deployment via the LangGraph Platform, enabling scalable, stateful agent workflows with visual prototyping and centralized management.

**Remember:**
- Agent orchestration
- Long-term memory
- Human-in-the-loop
- Streaming support
- Subgraphs and branching
- Checkpointing and persistence
