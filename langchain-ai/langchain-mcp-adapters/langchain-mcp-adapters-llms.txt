# Langchain Mcp Adapters

> The purpose of this project is to enable LangChain and LangGraph agents to interact with MCP-compliant tools via lightweight adapters. It facilitates the conversion of MCP tools into LangChain-compatible tools, allowing users to leverage external services (e.g., math or weather APIs) through a unified agent framework. The project also supports advanced use cases like multi-server connections and streaming HTTP transports, making it suitable for both local and distributed agent environments.

**Remember:**
- Anthropic Model Context Protocol (MCP)
- LangChain tools
- LangGraph agents
- MultiServerMCPClient
- Streamable HTTP transport
- ClientSession
