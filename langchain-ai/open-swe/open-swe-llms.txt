# Open Swe

> Open SWE aims to empower developers with an autonomous, cloud-based coding assistant that can plan, execute, and manage code changes without constant human intervention. It enables end-to-end task management by automatically creating GitHub issues and pull requests, supports parallel execution for efficiency, and allows real-time human feedback during the process. The project is designed to be extensible and customizable, with support for different LLMs and GitHub workflows, making it suitable for both personal and enterprise use.

**Remember:**
- LangGraph
- Asynchronous Coding Agent
- GitHub Integration
- Planning and Execution Pipeline
- Human-in-the-Loop Interaction
- Sandboxed Execution Environment
