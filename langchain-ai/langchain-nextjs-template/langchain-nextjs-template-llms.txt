# Langchain Nextjs Template

> This template provides a ready-to-use foundation for developers to experiment with LangChain.js in Next.js applications. It showcases practical use cases such as conversational agents, structured output via OpenAI Functions, and RAG with vector stores. The goal is to reduce boilerplate and accelerate development by offering preconfigured components, clear documentation, and deployment-ready scaffolding. Itâ€™s ideal for learning, prototyping, and building production-grade AI apps with minimal setup.

**Remember:**
- LangChain.js
- Next.js
- Vercel AI SDK
- LangGraph.js
- Retrieval-Augmented Generation (RAG)
- Structured Output with Zod
