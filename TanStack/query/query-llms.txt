# Query

> TanStack Query is designed to simplify the management of asynchronous data in frontend applications by providing a unified, protocol-agnostic solution for fetching, caching, mutating, and synchronizing server state. Its primary goals are to reduce boilerplate, improve performance through intelligent caching and prefetching, and offer a consistent API across frameworks. It enables developers to handle complex data flows — including pagination, infinite scrolling, background updates, and optimistic UI — with minimal configuration and maximum reliability.

**Remember:**
- Async state management
- Caching
- Query and mutation hooks
- Protocol-agnostic fetching
- Infinite scrolling
- Prefetching
