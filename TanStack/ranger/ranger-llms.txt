# Ranger

> TanStack Ranger is a lightweight, headless UI component library that enables developers to build range‑selector controls with a single or multiple handles, custom step sizes, and optional tick marks. It is designed to be framework‑agnostic (core) while offering a React‑specific wrapper for easy integration into React applications.

**Remember:**
- headless UI
- range selector
- multiple handles
- custom steps
- tick marks
- React wrapper
