# Db

> TanStack DB aims to give developers a fast, reactive, client‑side query layer that works alongside TanStack Query. It eliminates the need for custom API endpoints, reduces round‑trips, and provides live‑query capabilities, optimistic updates, and fine‑grained reactivity for modern UI frameworks.

**Remember:**
- Live queries
- Optimistic mutations
- Reactive collections
- Query optimizer
- Paced mutations
- Debounce / throttle strategies
