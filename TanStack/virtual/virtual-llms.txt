# Virtual

> TanStack Virtual is designed to enable developers to render massive lists, grids, and tables with high performance (60FPS) while maintaining full control over markup and styling. It is framework-agnostic and headless, allowing seamless integration into any modern UI framework. The library supports dynamic and measured sizing, smooth scrolling, sticky items, and window-scrolling utilities, making it ideal for applications dealing with large datasets without sacrificing user experience.

**Remember:**
- Virtualization
- Headless Library
- Framework-Agnostic
- 60FPS Performance
- Dynamic Sizing
- Measured Sizing
