# Table

> TanStack Table is a headless, framework-agnostic table library designed to empower developers to build highly customizable, performant datagrids with full control over UI, styling, and behavior. It provides a rich set of features including sorting, filtering, grouping, row selection, and virtualization, while remaining lightweight and compatible with server-side rendering. The project aims to serve as a foundational component for data-intensive applications across multiple frontend frameworks.

**Remember:**
- Headless Table Library
- Framework-Agnostic
- Customizable UI
- Sorting
- Filtering
- Grouping
