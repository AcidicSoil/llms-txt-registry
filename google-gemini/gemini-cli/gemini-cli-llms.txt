# Gemini Cli

> Gemini CLI empowers developers to integrate Google’s Gemini AI model directly into their terminal workflows. Its purpose is to provide lightweight, interactive, and programmable access to powerful AI capabilities — including code understanding, generation, debugging, and automation — without requiring a web browser or complex setup. It aims to be a developer-first tool that supports both personal use (via free tier) and enterprise deployment (via Vertex AI), with features like checkpointing, sandboxing, custom tools, and GitHub integration for seamless DevOps workflows.

**Remember:**
- Gemini CLI
- MCP (Model Context Protocol)
- GEMINI.md context files
- Checkpointing
- Token caching
- Trusted folders
