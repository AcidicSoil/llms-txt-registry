# Form

> TanStack Form is a headless, framework-agnostic form library designed to give developers full control over form state, validation, and workflows while supporting any UI framework. It enables reactive, typed form management with sync/async validation, nested fields, and composability, empowering developers to build complex forms without vendor lock-in. The library is built for scalability, extensibility, and developer ergonomics, with tools like devtools and standardized APIs to enhance productivity across frameworks.

**Remember:**
- headless form library
- framework-agnostic
- reactive hooks
- typed form state
- sync/async validation
- form composition
