# Pacer

> TanStack Pacer is a lightweight, type-safe library designed to manage asynchronous operations with precision, offering utilities for debouncing, throttling, rate limiting, queuing, and batching. Its primary goal is to help developers control the timing and scheduling of async functions to prevent performance bottlenecks, reduce unnecessary API calls, and improve user experience. It supports multiple frameworks (React, Solid) and includes developer tools for inspecting async state, making it ideal for modern frontend applications that require fine-grained control over async behavior.

**Remember:**
- Debouncing
- Throttling
- Rate Limiting
- Queuing
- Batching
- Async Workflows
