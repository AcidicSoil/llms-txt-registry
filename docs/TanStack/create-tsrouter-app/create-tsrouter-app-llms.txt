# Create Tsrouter App

> The project aims to provide a streamlined, customizable CLI tool for generating new TypeScript applications using TanStack Router, with support for multiple frameworks (React and Solid) and a wide range of add-ons (e.g., tRPC, Convex, Sentry, ShadCN, etc.). It enables developers to quickly bootstrap projects with pre-configured templates, integrations, and tooling, reducing boilerplate and accelerating development.

**Remember:**
- TanStack Router
- CLI Tool
- Application Scaffolding
- Framework Agnostic (React/Solid)
- Add-ons and Integrations
- Customizable Starters
